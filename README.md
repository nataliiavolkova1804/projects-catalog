Каталог проектов компании

Веб-приложение для просмотра и поиска проектов компании с удобным интерфейсом.

Функционал

- Поиск проектов — мгновенный поиск по названию, описанию и технологиям
- Фильтрация — по категориям (Веб-разработка, Мобильная разработка, AI/ML и др.)
- Сортировка — по имени (А-Я/Я-А) и году (новые/старые)
- Модальные окна — подробная информация о проекте при клике на карточку
- Адаптивный дизайн — корректное отображение на всех устройствах (от 360px до 1920px)
- Счётчик результатов — отображение количества найденных проектов

Технологии

- HTML5 — семантическая разметка
- CSS3 — стилизация, градиенты, анимации, медиа-запросы
- JavaScript — асинхронная загрузка данных, фильтрация, сортировка
- JSON — хранение данных о проектах

Структура проекта

projects-catalog/
├── index.html          # Главная страница
├── style.css           # Стили с адаптивностью
── app.js              # Логика приложения
├── data.json           # Данные о 20 проектах
└── images/             # Изображения проектов

Особенности дизайна

- Градиентный фон (фиолетово-синий)
- Карточки с тенями и hover-эффектами
- Плавные анимации
- Удобная навигация
- Современный UI/UX

Адаптивность

Проект полностью адаптирован под:
- Десктопы (1920px+)
- Планшеты (768px - 1024px)
- Мобильные устройства (360px - 768px)

Установка

1. Клонируйте репозиторий
2. Откройте index.html в браузере
3. Готово!
